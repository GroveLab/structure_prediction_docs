AlphaFold3

AlphaFold3, developed by DeepMind, predicts the 3D structures of proteins and other biological molecules with high accuracy. It is a significant advancement over AlphaFold2, capable of modeling interactions between proteins, nucleic acids, small molecules, and ions. AlphaFold3 introduces the "Pairformer" architecture and a diffusion-based sampling strategy to iteratively refine structure predictions. It assembles MSAs locally using Jackhmmer, allowing for improved accuracy in sequence alignment.

GitHub Repository: https://github.com/google-deepmind/alphafold3

ColabFold-AlphaFold2

ColabFold-AlphaFold2 is a community-developed implementation of AlphaFold2 that enhances accessibility by allowing users to run structure predictions efficiently via Google Colab or local installations. It retains the core deep-learning-based structure prediction capabilities of AlphaFold2 while using MMseqs2 for rapid MSA generation. This implementation optimizes computational efficiency and enables faster predictions, making it a widely used alternative for researchers needing scalable protein modeling solutions.

GitHub Repository: https://github.com/YoshitakaMo/localcolabfold

Boltz-1

Boltz-1 is an open-source AI model developed by MIT researchers that predicts the 3D structures of proteins, RNA, DNA, and small molecules. It handles modified residues, covalent ligands, and glycans, and can condition generation on pocket residues. Boltz-1 matches or outperforms AlphaFold3 in performance. It uses the MMseqs2 server for MSA generation.

GitHub Repository: https://github.com/jwohlwend/boltz

Chai-1

Chai-1 is another AlphaFold3-like model, but it can utilize both MSAs and protein language model embeddings (derived from ESM-2) as inputs. It employs the MMseqs2 server for MSA generation.

GitHub Repository: https://github.com/chaidiscovery/chai-lab

ESMFold

ESMFold is a protein structure prediction model developed by Meta AI. It predicts protein structures using a large language model trained on protein sequences without relying on MSAs.

GitHub Repository: https://github.com/facebookresearch/esm
