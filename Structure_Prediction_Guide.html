Structure Prediction Models: Usage Guide

AlphaFold3

AlphaFold3, developed by DeepMind, predicts the 3D structures of proteins and other biological molecules with high accuracy.

Detailed Instructions

ColabFold-AlphaFold2

ColabFold-AlphaFold2 is a community-developed implementation of AlphaFold2 that enhances accessibility by allowing users to run structure predictions efficiently via Google Colab or local installations.

Detailed Instructions

Boltz-1

Boltz-1 is an open-source AI model developed by MIT researchers that predicts the 3D structures of proteins, RNA, DNA, and small molecules.

Detailed Instructions

Chai-1

Chai-1 is another AlphaFold3-like model, but it can utilize both MSAs and protein language model embeddings (derived from ESM-2) as inputs.

Detailed Instructions

ESMFold

ESMFold is a protein structure prediction model developed by Meta AI. It predicts protein structures using a large language model trained on protein sequences without relying on MSAs.
